java 开发平台

为了以后开发的统一性，易维护性。部门内部关于java的开发应该采取统一的标准。

JDK

  • jdk 1.6

Servlet 容器

  • Tomcat 7

Servlet 版本

  • Servlet 3.0

Maven

  • Maven 3
  • 所有的项目都应该是Maven项目。

Java 开源框架/开源包

  • 针对同一个包，在不同的产品或项目中我们应该依赖同一个版本。
  • 建立一个pom项目，定义出常用开源包的版本和一些共有的配置。
  • 所有的java项目都应该继承这个项目。
