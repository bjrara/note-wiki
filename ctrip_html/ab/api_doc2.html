RuleType查询添加

  1. /ab/rule-types POST 新建一个 RupeType
  2. /ab/rule-types GET 列出所有的RupeType
  3. /ab/rule-types/{ruleTypeId} GET 获取指定的 RupeType
  4. /ab/rule-types/{ruleTypeId} POST 部分修改RupeType
  5. /ab/rule-types/{ruleTypeId} PUT 刷新整个RupeType
  6. /ab/rule-types/{ruleTypeId} DELETE 删除一个 RupeType

添加管理Experiment

  1. /ab/experiments POST 新建一个 Experiment
  2. /ab/experiments GET 列出所有的Experiment
  3. /ab/experiments/{experimentId} GET 获取指定的 Experiment
  4. /ab/experiments/{experimentId} POST 部分修改Experiment
  5. /ab/experiments/{experimentId} PUT 刷新整个Experiment
  6. /ab/experiments/{experimentId} DELETE 删除一个 Experiment

添加管理Alternative

  1. /ab/experiments/{experimentId}/alternatives POST 新建一个 Alternative
  2. /ab/experiments/{experimentId}/alternatives GET 列出所有的Alternative
  3. /ab/experiments/{experimentId}/alternatives/{alternativeId} GET 获取指定的 Alternative
  4. /ab/experiments/{experimentId}/alternatives/{alternativeId} POST 部分修改Alternative
  5. /ab/experiments/{experimentId}/alternatives/{alternativeId} PUT 刷新整个Alternative
  6. /ab/experiments/{experimentId}/alternatives/{alternativeId} DELETE 删除一个 Alternative

获取修改Alternative的参数

  1. /ab/experiments/{experimentId}/alternatives/{alternativeId}/rule-params GET 获取指定的 Alternative的规则参数。
  2. /ab/experiments/{experimentId}/alternatives/{alternativeId}/rule-params POST 部分修改Alternative的规则参数。
  3. /ab/experiments/{experimentId}/alternatives/{alternativeId}/rule-params PUT 刷新整个Alternative的规则参数。
