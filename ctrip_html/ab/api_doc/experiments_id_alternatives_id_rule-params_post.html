CTRIP

/ab/experiments/{experimentId}/alternatives/{alternativeId}/rule-params POST

功能

修改一个候选方案 Alternative 的规则参数

URL

/ab/experiments/{experimentId}/alternatives/{alternativeId}/rule-params

HTTP请求方式

POST

请求类型

FORM

请求参数

  • 路径参数
参数名 值 描述
experimentId String 要修改的 Alternative 所属的 Experiment
alternativeId String 要修改的 Alternative
  • Form 类型
参数名 值   描述
name String required Alternative 名称
description String   Alternative描述
isActive boolean:true/false   是否被激活
ruleParam1 String   分配参数一
ruleParam2 String   分配参数二
ruleParam3 String   分配参数三
ruleParam4 String   分配参数四
ruleParam5 String   分配参数五
ruleParam6 String   分配参数六
ruleParam7 String   分配参数七
ruleParam8 String   分配参数八
ruleParam9 String   分配参数九

返回结果

状态 结果
200 修改成功

测试

curl -F ruleParam1="30" https://baseurl/experiments/{experimentId}/alternatives/{alternativeId}/rule-params
