CTRIP

/ab/experiments/{experimentId} POST

功能

修改一个`Experiment`实例

URL

/ab/experiments/{experimentId}

HTTP请求方式

POST

请求类型

FORM

请求参数

  • Form 类型
参数名 值 描述
name String Experiment 名称
description String Experiment 描述
startDate YYY-mm-dd HH:MM 开始运行的时间
endDate YYY-mm-dd HH:MM 停止运行的时间
isActive boolean:true/false 是否被激活
ruleType String 采用的 RuleType 编码

返回结果

状态 结果
200 修改成功

测试

#Stop a experiment
curl -F isActive=false https://baseurl/ab/experiments/7
