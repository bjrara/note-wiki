CTRIP

/ab/experiments/{experimentId}/alternatives GET

功能

返回一个测试 Experiment 实例添所拥有的所有的候选方案 Alternative

URL

/ab/experiments/{experimentId}/alternatives

HTTP请求方式

GET

请求类型

无

请求参数

  • 路径参数
参数名 值 描述
experimentId String 要查询的 Experiment

返回格式

JSON XML
