CTRIP

/ab/rule-types/{ruleTypeId} PUT

功能

更新一种分配规则 RuleType

URL

/ab/rule-types/{ruleTypeId}

HTTP请求方式

PUT

请求类型

JSON XML

请求参数

  • XML 类型 
     <ruleType>
    <code>PERCENT</code>
    <description>group the user by percent.</description>
</ruleType>
  • JSON 类型 
    {
    "code": "PERCENT",
    "description": "group the user by percent."
}

注意: code信息并不会修改

返回结果

状态 结果
200 更新成功

测试

curl -v -X put -HContent-type:application/xml --data "<xml data>" https://baseurl/rule-types/2
curl -v -X put -HContent-type:application/json --data "<json data>" https://baseurl/rule-types/2
