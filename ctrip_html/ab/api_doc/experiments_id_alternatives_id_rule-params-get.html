CTRIP

/ab/experiments/{experimentId}/alternatives/{alternativeId}/rule-params GET

功能

返回一个指定的候选方案 Alternative 的分配规则参数

URL

/ab/experiments/{experimentId}/alternatives/{alternativeId}/rule-params

HTTP请求方式

GET

请求类型

无

请求参数

  • 路径参数
参数名 值 描述
experimentId String 要查询的 Experiment
alternativeId String 对应的 Alternative

返回格式

JSON XML
