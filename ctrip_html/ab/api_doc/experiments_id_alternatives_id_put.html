CTRIP

/ab/experiments/{experimentId}/alternatives/{alternativeId} PUT

功能

更新一个候选方案 Alternative

URL

/ab/experiments/{experimentId}/alternatives/{alternativeId}

HTTP请求方式

PUT

请求类型

JSON XML

请求参数

  • 路径参数
参数名 值 描述
experimentId String 要更新的 Alternative 所属的 Experiment
alternativeId String 要更新的 Alternative
  • XML 类型 
                <alternative>
                <code>A_A</code>
                <name>Alternative a</name>
                <description>test alternative a</description>
                <isActive>true</isActive>
                <ruleParams>
                    <param1>30</param1>
                </ruleParams>
            </alternative>
  • JSON 类型 
                {
                "code": "A_A",
                "name": "Alternative a",
                "description": "test alternative description a",
                "isActive": true,
                "ruleParams": {
                    "param1": "30",
                    "param2": null,
                    "param3": null,
                    "param4": null,
                    "param5": null,
                    "param6": null,
                    "param7": null,
                    "param8": null,
                    "param9": null
                }
            }
}

返回结果

状态 结果
200 更新成功
